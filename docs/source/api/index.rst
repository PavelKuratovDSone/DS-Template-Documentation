Engine
============================

Большинство функций можно получить через Engine

Текущие возможности шаблона

* Работа с монетами и алмазами
* Вывод debug информации на экран
* Инжектор зависимостей
* Собственная система ивентов
* Система исключений(ошибок)
* Анализ ввода игрока
* Система менеджеров(таймер, система состояния игры)
* Математические операции
* Физические надстройки
* Работа с сохранениями
* Работа с настройками
* Работа с внутриигровым магазином
* Расширения для работы с UI
* Работа с камерами Cinemashine
* Другие утилиты
